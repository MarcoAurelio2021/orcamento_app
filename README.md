# App de Orçamentos Flutter

Projeto Flutter pronto com:

- Novo orçamento
- Histórico de orçamentos
- Gerenciar tipos de serviço
- Dados da empresa
- Relatórios com PDF e compartilhamento de texto
- Persistência local com `shared_preferences`

## Como rodar

```bash
flutter pub get
flutter run
```

## Observação

Se você quiser gerar as pastas nativas padrão (android/ios/web/windows/linux/macos) no seu ambiente, rode:

```bash
flutter create .
```

Esse comando não sobrescreve seu `lib/` se você responder com cuidado ao conflito, mas em um projeto novo costuma apenas complementar a estrutura que estiver faltando.
